🚀 Quick Start Guide

📦 Prerequisites

```
🌐 Modern Web Browser (Chrome/Firefox/Safari/Edge)
💻 JavaScript Enabled
📱 Internet Connection (for fonts only)
```

⚙️ Installation Steps

```bash
# OPTION 1: Direct Use (No Installation)
1. Download the HTML file
2. Double-click to open in browser
3. Start calculating immediately!

# OPTION 2: Self-Hosting
git clone https://github.com/momonpxl/scor-point.git
cd scor-point
# Open index.html in your browser

# OPTION 3: Embed in Website
<iframe src="scor-point/index.html" width="100%" height="800px"></iframe>
```

🔑 First Time Setup

1. Fill Event Details
   · Event Name (e.g., "Weekly Scrim")
   · Date (auto-filled to today)
   · Organizer Name (your branding)
   · Creator Info (transparency)
2. Configure Point System
   · Adjust placement points (#1-15)
   · Set kill value (default: 1)
   · Reset to defaults anytime
3. Add Teams
   · Minimum 1 team
   · No maximum limit
   · Edit names anytime
4. Set Matches
   · Choose number of matches
   · Fill positions & kills
   · Auto-save enabled

⚠️ Important Notes

✅ Best Practices

```
• Fill all match data before calculating
• Double-check position numbers
• Use consistent team names
• Export CSV after each session
• Clear localStorage if switching events
```

❌ Common Mistakes

```
• Forgetting to set match count first
• Leaving kills empty (auto = 0)
• Not saving team name changes
• Overwriting data without export
```

🛡️ Data Safety

```
• All data stored LOCALLY only
• No internet transmission
• No tracking or analytics
• No cookies used
• Export before closing browser
```
